Menghitung Luas dan Keliling Lingkaran

      #Flow
      1. mengetahui jari2 atau diameter 
      2. mengetahui nilai dari phi 
      3. rumus luas lingkaran adalah phi x r x r 
      4. rumus keliling lingkaran adalah 2 x phi x r
    
      #Pseudocode 
      a. Luas Lingkaran
         1. input jari2 bertipe numberic 
         2. nilai phi adalah 3,14
         3. membuat variable hasilLuas bertipe numberic
         4. memasukan input jari2
         5. menghitung luas lingkaran dengan memasukan inputan jari2 ke rumus
         6. rumus luas lingkaran phi x r x r 
         7. memasukan hasil dari perhitungan luas lingkaran ke variable hasilLuas
         8. menampilkan variable hasilLuas
      
      b. Keliling Lingkaran
         1. input jari2 bertipe numberic
         2. nilai phi adalah 3,14
         3. membuat variable hasilKeliling bertipe numberic
         4. memasukan input jari2
         5. menghitung keliling lingkaran dengan memasukan inputan jari2 ke rumus
         6. rumus keliling lingkaran 2 x phi x r 
         7. memasukan hasil dari perhitungan luas lingkaran ke variable hasilKeliling
         8. menampilkan variable hasilKeliling
